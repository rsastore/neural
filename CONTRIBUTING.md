# Contributing to RSA Agentic

Thanks for your interest! Here's how to contribute:

## Getting Started

1. Fork the repo
2. Clone your fork
3. Create a branch: `git checkout -b feature/my-feature`
4. Make your changes
5. Test: `python3 -m py_compile neural.py agent.py planner.py`
6. Commit: `git commit -m "Description of change"`
7. Push: `git push origin feature/my-feature`
8. Open a PR

## Code Style

- Python 3.11+ type hints
- 4-space indentation
- Keep functions focused (one thing, well)
- Add docstrings for public APIs
- No external dependencies without discussion

## What to Work On

Check the Issues tab for open bugs and feature requests.
Or check the CHANGELOG for planned features.

## Questions?

Open a Discussion or Issue.
